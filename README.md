## Featured Project
### [Credit Card Fraud Detection System](https://github.com/samratsarkar624/fraud-detection-ml-api)
- This project is an end-to-end Fraud Detection System built using Machine Learning and deployed via a Flask web application.

It detects whether a transaction is fraudulent or normal based on input features, handling a highly imbalanced dataset (~0.17% fraud cases).
 
- Technologies used: `Python | Numpy | Scikit-Learn | Joblib | Flask API`
